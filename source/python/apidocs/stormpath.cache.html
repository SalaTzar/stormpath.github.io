manager Module¶

Cache manager abstraction.

class stormpath.cache.manager.CacheManager¶

Bases: object

Handles all the different caches used by the SDK

It keeps track which resource belongs to which cache. E.g stormpath.resources.directory.Directory resource data is stored in stormpath.cache.memory_store.MemoryStore while stormpath.resources.application.Application resource data is stored in stormpath.cache.redis_store.RedisStore. The CacheManager, along with stormpath.http.HttpExecutor is a part of the stormpath.data_store.DataStore.
